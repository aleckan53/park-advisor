# Park Advisor

[NPS api](https://www.nps.gov/subjects/digital/nps-data-api.htm) based, US national parks search app.
Users get detailed park info (photos, location, weather, entarance fees...) on one page. 

## Built with

* HTML/CSS
* JavaScript/JQuery
